# Remote-Local-Data
Android app to handle both remote and db with RxJava2, Retrofit2, Room, Java

This app will help you handle data from different resources like remote, local and shared prefernces. How we leverage RxJava operators
such as Merge & MergeDelayError (suggest to understand before using it ) from this link 
http://reactivex.io/documentation/operators/merge.html
